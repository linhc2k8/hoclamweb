    • PHP Wordpress

    This guide will cover installing Wordpress on a remote server. This will not configure wordpress, but you will have a working Wordpress installation ready for configuration via the WebUI. If you're familiar with PHP Wordpress, you know that it has the following requirements: Apache Web Server MySQL Database Server PHP To have a fully functional Wordpress installation, our Chef cookbook will also need to: Download the latest Wordpress package Configure Apache, MySQL, and PHP to work in harmony You could easily extend this recipe, automatically deploying blog posts and setting up configurations.
